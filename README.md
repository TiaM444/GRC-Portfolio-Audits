GRC Portfolio: Security Audits & Risk Categorizations
Author: Tia Mitchell, GRC Analyst

This repository contains a collection of governance, risk, and compliance projects focused on federal standards, identity management, and risk mitigation strategies.


📁 Featured Projects

1. First Citizens Bank: System Security Categorization
Focus: FIPS 199 Categorization for an Integrated Automation Platform.
Key Outcome: Determined a High-Water Mark (High) risk level based on the sensitivity of administrative accounts and vulnerability data.

2. Carraway IT: User Access Control Audit
Focus: IAM Lifecycle and Quarterly Access Reviews.
Key Outcome: Identified five high-risk findings, including orphaned accounts and privilege creep, resulting in a Needs Improvement rating.

🛠️ Technical Skills Demonstrated
Frameworks: FIPS 199, NIST SP 800-53, ISO 27001.
Audit Areas: Identity & Access Management (IAM), Role-Based Access Control (RBAC), and Principle of Least Privilege (PoLP).
Risk Management: Root Cause Analysis and Management Action Planning.
